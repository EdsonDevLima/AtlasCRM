# Atlas CRM Frontend

Aplicação frontend do Atlas CRM construída com React, TypeScript e Vite.

## Comandos principais

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Variáveis de ambiente

Crie ou ajuste o arquivo `.env`:

```env
VITE_API_URL=http://localhost:3000
VITE_APP_NAME=Atlas CRM
```
